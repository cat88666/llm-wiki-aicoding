# Spec 模板（SDD 用）

> 用法：动手让 AI 写代码前，先和它一起把这张表填出来，**你审一遍、改两处**，再让它写。
> 你审的是这张表（十几行），不是它写的实现（上百行）。

## 一句话目标
> 这个东西要解决什么、做成什么样算对。

## 数据模型
| 字段 | 类型 | 约束（必填？范围？默认？）|
|---|---|---|
| | | |

## 端点 / 接口
| 动作 | 输入 | 成功返回 | 失败情况（状态码）|
|---|---|---|---|
| | | | |

## 边界（每条之后会变成一条测试）
- 空 / 极值 / 重复 / 不存在 / 冲突……
