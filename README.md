# llm-wiki-aicoding

AI Coding 知识库。`raw/` 保存原始课程材料，`wiki/` 按 Karpathy wiki 的方式收敛为少量可复用页面类型。

## Wiki

- [Concepts](wiki/concepts/README.md)：可复用概念，如 SDD、TDD、特征测试、上下文工程。
- [Summaries](wiki/summaries/README.md)：主题总结，如 AI 可靠编程、大项目安全重构、工程化治理。
- [Synthesis](wiki/synthesis/README.md)：跨课程综合方法，如工程闭环、30 天落地路线、翻车处置。

## 摄入原则

```
raw 原文
  └── concepts：抽出稳定概念
        └── summaries：按主题组织学习地图
              └── synthesis：沉淀可执行方法论
```
